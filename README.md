# Create Warlock

This package is used to generate a [Warlock.js](https://warlock.js.org) project.

## Usage

Run the following command:

```bash
npx create-warlock
```

Or

```bash
yarn create warlock
```

Or

```bash
pnpm create warlock
```

Then follow the instructions, it is easy as that!
